# Fair Use Doctrine

The fair use doctrine allows limited use of copyrighted material without permission for purposes such as criticism, commentary, news reporting, education, or research.

## Types of Fair Use

### Classic Fair Use

Using a trademark to describe goods or services **without indicating their source**.

**Purpose**: Help prevent monopolies on descriptive terms.

**Example**: "iPhone-compatible charger" (describing compatibility, not claiming to be Apple).

### Nominative Fair Use (Most Common)

Allows use of another's mark to identify their goods/services when they cannot be easily identified otherwise.

**Example**: Media outlets referring to trademarked names for reporting purposes (provided there's no false affiliation suggested).

## When Does Fair Use Apply?

Fair use applies when the mark is used in **good faith** without causing **consumer confusion**.

## Requirements for Fair Use

### Fair Trademark Usage

To qualify as fair trademark use:

1. **Necessary for Description**
   - Use of plaintiff's mark must be necessary to describe goods/services.

2. **Minimal Use**
   - Only amount necessary for description should be used.

3. **Accurate Relationship**
   - Usage must reflect true relationship between user and plaintiff.

### Nominative Trademark Usage

To qualify as nominative use:

1. **Accurate Reference**
   - Must accurately refer to TM owner or goods/services.
   - Cannot be misleading or defamatory.

2. **No Implied Endorsement**
   - Must not suggest endorsement/sponsorship by TM owner.

3. **No Easier Alternative**
   - Should be no easier way to refer to owner or products.

4. **Minimal Use**
   - Only amount necessary to identify TM owner.
