# Messaging Applications

To ensure we can assist you effectively with reporting messengers, please include a clear statement indicating your intention.

{% callout type="tip" %}
**Be Specific**: Specify if you would like to take down a WhatsApp account, Telegram channel, or Discord server. Provide the URL or identifier of the account.
{% /callout %}

These should be reported as **"Social"** incidents.

## Supported Messaging Platforms

- [WhatsApp →](whatsapp)
- [Telegram →](telegram)
- [Discord →](discord)

## General Requirements

All messaging platform takedowns require:

✅ **Letter of Authorization (LOA)**
✅ **Clear identification** of the infringing account/channel/server
✅ **Screenshots** showing the infringement with timestamps
✅ **Context** explaining the violation
