# Browser Extensions

## Evidence Requirements

✅ **Clear Evidence of Copyright Infringement**

Such as:
- Screenshots showing brand misuse (logo, brand name).
- Screenshots of the extension in action (login form, phishing behavior).
- Link to the extension in Chrome/Edge/Firefox store.

✅ **Proof of Unauthorized Use**
- Show the extension uses proprietary code without authorization.
- Demonstrate it mimics official functionality.
- Prove consumer confusion.

✅ **LOA**

## Best Practices

✅ Show the extension in action.
✅ Provide link to official extension (if you have one).
✅ Document user complaints or confusion.
✅ Include store listing URL.
