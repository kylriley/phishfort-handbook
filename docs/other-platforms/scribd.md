# Scribd Takedowns

## Evidence Requirements

✅ **Name of the Company Being Impersonated**

✅ **Suspicious URLs** (most important)
- Direct links to infringing documents.

✅ **File Titles**
- According to Scribd: "Enter titles of copyrighted material being infringed upon and, if possible, additional identifying information such as ISBNs, publication dates, etc—or, if the material is a web page, the URL where the original can be found."

✅ **TM Registration Numbers/Documents**
- Especially helpful when sub-brands are targeted.
